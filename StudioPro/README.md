# AI Template Studio Pro

AI Template Studio Pro is a high-performance web-based IDE designed specifically for creating, testing, and managing AI templates.

## Key Features

- **Code Editor Integration:** Uses CodeMirror for a professional-grade code editing experience for HTML, CSS, and JS.
- **AI-Powered Workflows:** Specifically built to streamline the generation and refinement of AI templates.
- **Reactive UI:** Built with Alpine.js for a smooth, responsive, and lightweight user experience.
- **Thumbnail Generation:** Integrated html2canvas for creating project previews and snapshots.
- **Mobile-Optimized:** Features specialized viewport fixes and safe-area adjustments for high-end mobile devices.
- **Modern Styling:** Leverages Tailwind CSS for clean and efficient interface design.

## Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JS.
- **Libraries:** Alpine.js, CodeMirror, Tailwind CSS, html2canvas, Font Awesome.
- **Icons:** Font Awesome 6.4.0.

## Getting Started

1. Open `StudioPro.html` in any modern web browser to access the studio.
2. Create or edit templates directly in the integrated editor.
3. Use the built-in tools for generating thumbnails and managing your projects.

## Project Structure

- `StudioPro.html`: Main application file containing the UI and core logic.
- `https://cdn.tailwindcss.com`: CDN-based styling.
- `https://cdnjs.cloudflare.com/ajax/libs/codemirror/`: Editor assets.
