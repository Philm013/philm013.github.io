Jira CSV Import & Help

Customizing for your Workflow

Go to Settings & Fields > Field Configuration to toggle fields like Resolution, Time Tracking, or Components to match your specific Jira configuration. Changes apply immediately.

CSV Import Guidelines

When exporting, the system adheres to the following Jira CSV standards for each field:

  • Summary: The issue title (Required).
  • Work Item Key: (e.g. PROJ-101). Used to update existing items if re-imported to Jira.
  • Components/Versions/Labels: Comma-separated values supported.
  • Time Tracking: Exported in Seconds (Original Estimate, Time Spent). The UI accepts "1h 30m" format.
  • Dates: Format is ISO 8601 (YYYY-MM-DD).
