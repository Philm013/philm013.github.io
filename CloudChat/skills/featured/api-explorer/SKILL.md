# Public API Explorer

## Instructions

Call the `explore_public_apis` tool with the following parameters:
- `category`: String (optional). Category to filter by (e.g. "Animals", "Finance", "Music", "Sports").
- `keyword`: String (optional). Keyword to search API names and descriptions.
