🧠 Gemma 4 — Native Assistant

What is this?

This is a fully private AI assistant powered by Google's Gemma 4 model. Everything runs entirely in your browser using WebGPU — no data ever leaves your device. No accounts, no cloud servers, no tracking.

How it works

The Gemma 4 E2B ONNX model (~1.5 GB) is downloaded once and cached locally in your browser via transformers.js. All inference runs on your device's GPU via WebGPU, providing fast, private AI that works even offline after the initial setup.

Requirements

  • Browser: Chrome 113+, Edge 113+, or any Chromium-based browser with WebGPU support
  • GPU: A WebGPU-compatible GPU (most modern GPUs from 2018+)
  • Storage: ~1.5 GB for the model + space for your data
  • RAM: 4 GB+ recommended for smooth operation

Privacy & Security

Your conversations, documents, and memories are stored locally in your browser's IndexedDB. Nothing is sent to any server. The only network requests are for optional features like web search (which you configure yourself) and the initial model download.

Agentic AI

The AI uses Gemma 4's native tool-calling format to autonomously decide when and how to use tools. It can chain multiple tool calls per message, reason through problems in its thinking channel, and provide comprehensive answers grounded in real data.

Conversation Management

  • Edit: Click ✏️ on any user message to edit and resend
  • Retry: Click 🔄 to regenerate an AI response
  • Delete: Remove individual messages or entire turns
  • Copy: Copy AI responses to clipboard with one click
  • History: All conversations are auto-saved and can be restored

Media Handling

You can attach images, audio files, and video files to your messages. These are displayed inline in the conversation for reference and review.

🖼️ Images

  • Upload: Click the 📎 button and select image files, or drag & drop images onto the chat
  • Paste: Paste images directly from your clipboard (Ctrl/Cmd+V)
  • Preview: Click any image thumbnail in a message to view it full-size in a lightbox
  • Formats: JPEG, PNG, GIF, WebP, SVG, and all browser-supported image formats
  • Context: The AI is informed about attached images and can discuss them based on filename and context

🎵 Audio

  • Upload: Attach audio files via the file picker or drag & drop
  • Playback: Audio files appear as inline players in your messages — play, pause, and scrub directly in the chat
  • Formats: MP3, WAV, OGG, AAC, FLAC, WebM audio, and all browser-supported audio formats
  • Context: The AI is informed about attached audio files and can reference them in the conversation

🎬 Video

  • Upload: Attach video files via the file picker or drag & drop
  • Playback: Videos appear as inline players with full controls (play, pause, volume, fullscreen)
  • Formats: MP4, WebM, OGG video, and all browser-supported video formats
  • Context: The AI is informed about attached videos and can discuss them in context

📄 Documents

  • Documents (PDF, DOCX, TXT, MD, CSV, JSON) are ingested into the semantic library for RAG (Retrieval-Augmented Generation)
  • Once ingested, the AI can search and reference document content to answer your questions
  • Manage your document library from the 📚 Library panel

⚠️ Multimodal Note

The Gemma 4 E2B model used in this app is a text-focused model. While you can attach and display media files in the conversation, the AI processes text-based descriptions of attachments rather than analyzing visual/audio content directly. For full multimodal analysis, consider using the LocalMind version which supports Gemma 4 multimodal models.

Available Tools

The AI has access to a growing library of tools it can use automatically. You can also manually insert tool calls using the 🛠️ button.

Built-in Tools

  • calculate — Evaluate math expressions (basic arithmetic to scientific notation)
  • web_search — Search the web via Brave, Tavily, or SearXNG (requires configuration)
  • search_media — Find images & videos from Unsplash or Pexels
  • fetch_page — Retrieve and read the content of any web page
  • query_wikipedia — Look up information from Wikipedia in 140+ languages
  • get_weather — Get current weather and forecasts for any location
  • store_memory — Save facts, preferences, or notes to persistent memory
  • search_memory — Recall stored information from your knowledge base
  • delete_memory — Remove specific memories or clear categories
  • interactive_map — Display an interactive map of any location
  • set_reminder — Set timed reminders that pop up in the chat
  • generate_qr — Generate QR codes for any text or URL
  • restaurant_roulette — Spin a fun wheel to pick a random restaurant nearby
  • mood_music — Generate or discover music based on mood or query

Slash Commands

  • /clear — Clear the current conversation
  • /settings — Open the settings panel
  • /tools — Show available tools
  • /memory — Open the memory panel
  • /batch — Open batch prompt mode

@ Mentions

Type @ in the chat input to search and reference your stored documents by name. The referenced document's content will be included as context for the AI.

Tips & Tricks

Keyboard Shortcuts

  • Enter — Send message
  • Shift + Enter — New line
  • Esc — Close menus & panels
  • Ctrl/Cmd + V — Paste images from clipboard

Getting Better Results

  • Be specific: Instead of "tell me about space", try "explain how black holes form, including Hawking radiation"
  • Use thinking mode: Keep thinking mode enabled (Settings) for complex questions — the AI reasons more carefully
  • Adjust temperature: Lower temperature (0.2-0.4) for factual/coding tasks, higher (0.8-1.2) for creative writing
  • Upload documents: Add reference materials to the Library — the AI will search them automatically
  • Chain requests: The AI can perform multiple tool calls per message, so feel free to ask complex questions

Troubleshooting

  • Slow responses? Reduce Max Tokens in Settings, or close other GPU-intensive tabs
  • Model won't load? Ensure you have WebGPU support (Chrome 113+) and try purging the cache. The ONNX model is downloaded and cached automatically by transformers.js
  • Repetitive output? Lower the temperature or clear the conversation history
  • Web search not working? Make sure you've configured a search provider and API key in Settings
  • Memory full? Use the Memory panel to audit and clean up duplicate entries

Gemma 4 Model Info (ONNX)

This app uses the ONNX variant of Gemma 4 E2B (E2B = Efficient 2 Billion) via transformers.js from Hugging Face. Key capabilities:

  • Reasoning and chain-of-thought via native thinking mode
  • Function/tool calling with structured outputs
  • 140+ language translation
  • 8K context window
  • Runs locally with ~1.5 GB VRAM
