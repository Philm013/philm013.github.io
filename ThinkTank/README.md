# ThinkTank - NGSS-Aligned AI Study Tool

ThinkTank is a powerful AI study tool designed to help students master science concepts through an NGSS-aligned curriculum.

## Key Features

- **NGSS Alignment:** All study content and AI interactions are aligned with Next Generation Science Standards.
- **AI-Powered Study Sessions:** Interactive AI-driven tutoring and study sessions.
- **Modern, Clean UI:** Designed with the Inter font family for high readability and a clean aesthetic.
- **Interactive Components:** Features polls, quizzes, and other interactive elements.
- **Responsive Design:** Optimized for both desktop and mobile use.

## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Styling:** Custom Tailwind-like styling for a modern look.
- **Fonts:** Google Fonts Inter for readability.

## Getting Started

1. Open `index.html` in any modern web browser to access the application.
2. Select your study mode or use the AI-powered study assistance.
3. Track your progress against the NGSS-aligned curriculum.

## Project Structure

- `index.html`: Main entry point and application logic.
- `https://fonts.googleapis.com/css2?family=Inter`: Modern typography.
