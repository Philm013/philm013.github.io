# Digital Synapse | Research Hub

Digital Synapse is a modern, responsive science news aggregator and research hub designed to keep users informed with the latest scientific discoveries and research.

## Features

- **Science News Aggregation:** Real-time updates on scientific news.
- **Research Hub:** A centralized interface for exploring research topics.
- **Dual Theme Support:** Fully integrated Dark and Light modes.
- **Mobile-First Design:** Optimized for a seamless experience across all devices, including mobile-specific viewport adjustments.
- **Interactive UI:** Smooth transitions and a clean, modern aesthetic.

## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Icons:** Font Awesome 6.4.0.
- **Styling:** Custom CSS variables for flexible theming.

## Getting Started

1. Open `ScienceNewsV2.html` in any modern web browser to access the latest version.
2. Toggle between light and dark modes using the UI controls.
