# Assessment Analyzer

The Assessment Analyzer is a client-side tool for reviewing and analyzing educational assessment materials against a rubric. It allows users to load PDF documents, highlight text, add comments, and generate reports.

This directory contains two versions of the tool:

*   **[K-12 Science Assessment Review](./index.html):** A tool for reviewing K-12 science assessments.
*   **[EQuIP Science Assessment & Design Review](./ALT/index.html):** A version of the tool tailored to the EQuIP (Educators Evaluating the Quality of Instructional Products) rubric.

Both versions are entirely client-side, using IndexedDB for local storage of all data.
