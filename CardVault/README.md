# CardVault

A powerful, browser-based sports card cataloging tool with AI-powered scanning.

## Features

- **AI-Powered Scanning:** Uses Gemini AI and OpenCV.js for high-accuracy card recognition and data extraction.
- **Computer Vision:** Real-time card detection and cropping using `opencv.js`.
- **Collection Management:** Organize your sports card collection with ease.
- **Glassmorphism UI:** A modern, sleek interface with a focus on ease of use.
- **Mobile-First Design:** Optimized for mobile devices with safe-area support.
- **PWA Ready:** Manifest included for a standalone app experience.

## Technologies Used

- **Tailwind CSS:** For modern styling.
- **Iconify:** For dynamic icon loading.
- **Chart.js:** For data visualization.
- **OpenCV.js:** For computer vision tasks (card detection and cropping).
- **Gemini AI:** For advanced card identification and metadata enrichment.
