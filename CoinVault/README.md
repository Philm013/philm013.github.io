# CoinVault

A sophisticated, browser-based coin and currency cataloging tool with AI-powered scanning.

## Features

- **AI-Powered Scanning:** Uses Gemini AI and OpenCV.js for high-accuracy coin recognition and data extraction.
- **Computer Vision:** Real-time coin detection and cropping using `opencv.js`.
- **Collection Management:** Organize your numismatic collection with precision.
- **Elegant UI:** A stone-themed, futuristic interface designed for collectors.
- **Mobile-First Design:** Optimized for mobile devices with safe-area support.
- **PWA Ready:** Manifest included for a standalone app experience.

## Technologies Used

- **Tailwind CSS:** For modern styling.
- **Iconify:** For dynamic icon loading.
- **Chart.js:** For data visualization.
- **OpenCV.js:** For computer vision tasks (coin detection and cropping).
- **Gemini AI:** For advanced coin identification and metadata enrichment.
