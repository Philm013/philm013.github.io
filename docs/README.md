# Documentation

This directory contains documentation files for various components and technologies used in the projects in this repository.

## Files

*   `GeminiJS.md`: Documentation related to a Gemini JavaScript library or SDK.
*   `GeminiRest.md`: Documentation related to the Gemini REST API.
*   `peerjs.md`: Documentation for PeerJS.
*   `tldraw_combined.md`: Combined documentation for tldraw.
*   `tldraw/`: A directory containing more specific documentation for the tldraw project.
*   `context.sh`: A shell script likely used for gathering context for documentation or development.
*   `get_md.sh`: A shell script for retrieving markdown files.
