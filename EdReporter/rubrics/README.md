# EdReporter Rubrics

This directory contains the JavaScript files that define the various rubrics used in the EdReports 2.0 Review Tool.

Each file in this directory defines a single rubric object and adds it to the global `window.rubrics` object so it can be loaded by the main application.

## Files

*   `ela-k2.js`: Rubric for ELA, grades K-2.
*   `mll-science-k12.js`: Rubric for MLL Science, grades K-12.
*   `science-68.js`: Rubric for Science, grades 6-8.
*   `science-912.js`: Rubric for Science, grades 9-12.
*   `science-k5.js`: Rubric for Science, grades K-5.
*   `rubric-template [No Evidence Guide].js`: A template for creating new rubrics.
