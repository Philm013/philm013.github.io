# ProximaNova (Project Nova v23)

ProximaNova is a high-performance, canvas-based strategic expansion game set in a futuristic space environment. Players engage in real-time management of buildings, units, and environmental challenges.

## Key Features

- **Multi-Layer Canvas Rendering:** High-performance graphics using separate canvases for terrain, buildings, dynamic units, and UI interactions.
- **Environmental Hazard System:** Dynamic gameplay events including dust storms, solar flares, and magnetic storms that impact strategy.
- **Real-Time Strategy (RTS) Elements:** Unit selection, movement, health management, and building placement.
- **Minimap & Event Log:** Strategic overview tools for tracking world state and historical events.
- **Responsive Interface:** Adaptive layout with specific optimizations for desktop and mobile environments.

## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Graphics:** Multi-canvas rendering for performance.
- **Mobile Support:** PWA-ready with manifest and theme-color support.

## Getting Started

1. Open `index.html` in a modern web browser to launch the game.
2. Use the on-screen controls to manage buildings and units.
3. Monitor the event log and alerts for incoming environmental hazards.

## Development

The project is structured with a modular approach:
- `css/style.css`: Core game styling and layout.
- `js/`: Modular game logic (renderer, unit management, environmental events).
- `index.html`: Main game container and UI structure.
