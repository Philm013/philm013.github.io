    Help & Information

    Layouts & Map Types

    • Mind Map & Sunburst: These layouts work best for tree-like structures (one parent per node).
    • Flowchart: This layout is for processes and can have nodes with multiple parents (e.g., merging paths).
    • Forcing a Layout: If you've edited a map and can no longer switch to a Mind Map view, you can right-click the canvas background and select "Set Layout to Mind Map" to override the automatic check.

    Desktop Controls

    • Pan: Click and drag the background.
    • Zoom: Use the mouse wheel.
    • Multi-Select (Lasso): Hold Shift and drag on the canvas.
    • Multi-Select (Click): Hold Ctrl (or Cmd on Mac) and click nodes.
    • Copy/Paste Selection: Use Ctrl/Cmd + C to copy and Ctrl/Cmd + V to paste.
    • Context Menu: Right-click on a node, link, or the canvas.

    Mobile / Touch Controls

    • Pan: Touch and drag the background.
    • Zoom: Pinch to zoom.
    • Multi-Select: Tap the Select Tool in the toolbar to enter Select Mode. In this mode, tap nodes to add/remove them from your selection. Tap the tool again to exit.
    • Context Menu: Long-press on a node or link.

    Primary AI Tools

    These are the main tools the AI uses to build your map.

    • research_and_add_node(topic, parentNodeId): This is the primary tool. It researches a topic, synthesizes the information into a detailed report, and creates a new node with that report in its details panel. Use this for adding any new concept to the map.
    • add_simple_node(label, parentNodeId): Adds a new node without any research. Use this for structural elements like "Pros", "Cons", or "My Ideas".
    • expandNodeWithAI(nodeId, instruction): Adds several new child nodes based on a topic, researching each one as it's created.
    • connectNodesById(sourceNodeId, targetNodeId): Creates a link between two nodes.
