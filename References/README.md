# References

Central reference hub for shared docs, implementation guidance, and external repo pointers used across this workspace.

## What Is In This Folder

- `JS_LIBRARIES_DOCUMENTATION.md`: Canonical JavaScript library matrix with official links, use cases, and quick how-to guidance.
- `JS-Libraries.html`: Browser-readable version of JavaScript library references.
- `REFERENCE_REPOS.md`: Curated list of repositories and examples worth reusing.
- `RepoReferences.html`: Browser-readable companion for repository references.

## How To Use This Folder

1. Start with `JS_LIBRARIES_DOCUMENTATION.md` when selecting or implementing a library.
2. Use `REFERENCE_REPOS.md` when you need proven implementation examples.
3. Open the `.html` versions when you want quick in-browser browsing/sharing.

## Contribution Rules

When adding or updating references:

1. Prefer official documentation links over blogs/aggregators.
2. Keep entries short and implementation-focused.
3. For new libraries, include the same three essentials: official link, when to use, and basic how to use.
4. Keep this README updated when files are added or renamed.
